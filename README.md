# Grafo AR - Sistema de Creación de Grafos por Realidad Aumentada

## Descripción General

Este proyecto implementa un sistema interactivo para la creación de grafos matemáticos utilizando **visión por computadora** y **detección de manos** en tiempo real. A través de gestos naturales capturados por la cámara web, el usuario puede construir grafos de manera intuitiva y visual.

### Estado Actual

✅ **Completado:**
- Creación de nodos mediante gesto de dedo índice (3 segundos quieto)
- Creación de aristas no dirigidas mediante gesto de pinza (índice + pulgar)
- Eliminación de nodos/aristas mediante gesto de puño (3 segundos quieto)
- Visualización en tiempo real con feedback visual
- Guardado de capturas (tecla 'S')
- Suavizado de movimiento (filtro EMA)
- Soporte completo para grafos no dirigidos

❌ **Pendiente:**
- Generación de matrices de adyacencia e incidencia
- Implementación de grafos dirigidos
- Exportación de matrices a archivo
- Visualización de matrices en interfaz
